Works with Claude Code, Factory Droid, Antigravity, Gemini, and standalone CLI.

### Interactive Menu
```bash

```

Browse all 41+ skills and commands with visual navigation.

```bash

prologue ls quality             # List quality category
```

### Show Banner
```bash

```

```bash

prologue install gemini         # Install for Gemini
```
