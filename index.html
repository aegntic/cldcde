Core NPM Packages

@aegntic/cldcde-cli-shortcuts

High-leverage commands and ergonomic aliases that reduce repetitive terminal operations for Claude Code workflows.

  • 40+ curated shortcuts
  • Fast setup for new environments
  • Consistent team workflows

@aegntic/cldcde-context-tracker

Live token/context visibility to prevent context loss and maintain coherent, long-running coding sessions.

  • Real-time context monitoring
  • Risk alerts before overflow
  • Actionable optimization hints

Install in under 60 seconds

# Install shortcuts globally
npm i -g @aegntic/cldcde-cli-shortcuts

# Run context tracker in your repo
npx @aegntic/cldcde-context-tracker

# Optional: combine with your Claude Code shell profile
cldcde setup --profile claude
