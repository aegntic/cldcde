## **Installation**

Each module can be installed independently:

```bash
# Install individual modules
npm install @sota/webassembly-module
npm install @sota/ai-features-module
npm install @sota/animation-module

# Or install all modules
npm install @sota/template-addons
```

## **Quick Start**

```javascript
// Import and initialize modules
import { WebAssemblyModule } from '@sota/webassembly-module';
import { AIFeaturesModule } from '@sota/ai-features-module';

// Initialize modules
const wasm = new WebAssemblyModule();
const ai = new AIFeaturesModule();

// Use module features
await wasm.loadImageProcessor();
await ai.initializePersonalization();
```
