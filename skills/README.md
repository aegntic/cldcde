# AE.LTD Skills in `aegntic/cldcde`

These skills are curated for Codex, Agent Zero, ZeroClaw, and ClawReform compatible loaders.

## Core Skills

- `ae-ltd-mcp-foundry` - Scaffold and harden MCP servers
- `ae-ltd-worktree-mesh` - Git worktree orchestration
- `ae-ltd-mutation-gate` - Mutation testing workflows
- `ae-ltd-claude-template-switchboard` - Template management

## Development Tools

- `ae-ltd-skill-builder` - Factory for creating, validating, and managing Claude Code skills with complete tooling (scripts, hooks, prompts, templates)

## Niche Hidden Gems

- `ae-ltd-context7-radar`
- `ae-ltd-visual-regression-forge`
- `ae-ltd-n8n-orbit`

Each skill folder includes:
- `SKILL.md` - Main skill definition
- `SKILL.toml` - Configuration metadata
- `agents/openai.yaml` - Agent interface

Advanced skills (like `ae-ltd-skill-builder`) also include:
- `scripts/` - Executable tooling
- `hooks/` - Lifecycle automation
- `prompts/` - System context extensions
- `docs/` - Detailed documentation
- `resources/` - Templates and examples
