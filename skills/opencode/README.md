# Opencode Skills

Custom skills adapted from Claude Code for use with opencode.

## Available Skills

| Skill | Description |
|-------|-------------|
| cloud-agent | Cloud agents with sandbox and recording capabilities |
| auto-demo | Automatic demo generation and recording system |
| bd-management | Beads workflow optimizer for task management |
| fileio-swarm-advanced | File I/O and shared state for multi-agent swarms |

## Usage

These skills should be copied to `~/.opencode/skills/` for global availability in opencode.

## Notes

- Most cldcde skills are already built into opencode (29+ skills)
- These 4 skills are unique to Claude Code and have been adapted for opencode
- Some skills require external dependencies (docker, bd, Node.js, etc.)
