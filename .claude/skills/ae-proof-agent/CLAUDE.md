# ae-proof-agent

Use this package in Claude sessions when doing product diligence, technical planning, or build-vs-buy analysis for the ae.ltd proof-agent concept.

Workflow:

1. Read `references/RESEARCH-GUARDRAILS.md`.
2. Read `references/COMPETITIVE-DILIGENCE.md`.
3. Use `references/PRD.md` and `references/TECHSPEC.md`.
4. Run `prompts/00-SYSTEM.md`, then `prompts/01-DISCOVERY.md` through `prompts/05-SYNTHESIS.md`.
5. Use `templates/SCORECARD.md` to compare outputs.
