# ae-proof-agent

Use this package as a diligence-grade prompt and specification bundle for the ae.ltd proof-agent concept.

Start here:

1. Read `references/RESEARCH-GUARDRAILS.md`.
2. Read `references/COMPETITIVE-DILIGENCE.md`.
3. Use `references/PRD.md` and `references/TECHSPEC.md` as the baseline.
4. Execute the prompt chain in `prompts/`.
5. Score outputs with `templates/SCORECARD.md`.

Rules:

- Separate `Verified`, `Inference`, and `Unknown`.
- Do not invent backend or infra details.
- Use exact dates for recent releases.
- Prefer primary sources and decision-useful analysis.
